module 2 : computer aided design

Subject : Report on computer aided design course thai is module two of the Fab-lab course.



The second module was about computer aided designing which taught us to design in 2D and 3D using softwares such as autocad and fusion 360. The first software that I was taught was fusion 360. Fusion 360 is cloud based platform accessed through web browser. The data lives in the central location (the cloud) for on demand access to system resources without user management and multi-platform communication. The second software was autocad, the basics were same as fusion 360 but there were some other resources like the command panel which made the usage more easy. Both softwares were autodesk software and were used by benifiting from the educational license for student.
